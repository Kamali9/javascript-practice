AWS CloudScape Open Source 2016 Responsiveness, Faster
Cloudscape consists of an extensive set of guidelines to create web applications, along with the design resources and front-end components to streamline implementation.
AWS released Cloudscape as open-source so that anyone building cloud products can benefit from Cloudscape design system.
It offers a solid base of 60+ components, 30+ pattern guidelines, and 20+ demos to make your work easier. To customize your user experience, Cloudscape offers theming, dark mode, and content density modes.
GOOGLE Material Design System Open Source 2014 Design like real.
Material Design is a design language developed by Google to help teams build high-quality digital experiences for Android, iOS, Flutter, and the web.
It was based on a grid-based system.
The main purpose of Material Design is the creation of a new visual language that combines principles of good design with technical and scientific innovation. Designer Matias Duarte explained that, "unlike real paper, our digital material can expand and reform intelligently. Material has physical surfaces and edges. Seams and shadows provide meaning about what you can touch." Google states that their new design language is based on paper and ink but implementation takes place in an advanced manner.
Microsoft Fluent Design System open-source 2017 Fluent Design System allows developers to build cross-platform applications with a unified design language.
Fluent Design System is based on five key elements: light, depth, motion, material, and scale.
These elements work together to create a visually appealing and engaging user experience. The system provides a set of guidelines, tools, and resources to help designers and developers create applications that are consistent with the Fluent Design language.
Atlassian Atlassian Design System semi open-source Atlassian Design System is based on several key principles, including simplicity, intuitiveness, and flexibility.
The system provides a set of components, including UI controls, icons, and typography, that can be used to build applications for various platforms, including web, mobile, and desktop.
open source:
AtlasKit: A set of open-source UI components licensed under Apache 2.0., AMPS (Atlassian Module Packaging System): An open-source tool for building and managing Atlassian plugins.
ACE (Atlassian Connect Express): An open-source framework for building Atlassian Connect apps.
AC Spring Boot: An open-source framework for building Atlassian Connect apps using Spring Boot.
Not open-source:
Atlassian Design System as a whole: While some components are open-source, the entire system is not.
Forge runtime libraries and toolchain: These are not currently open-source, but Atlassian has indicated that they may be open-sourced in the future.
Atlas Kit: While the source code is mirrored, it is not readily buildable or open for contributions.
